# 90_ARCHIVE — The History

> Deprecated, completed, and historical documents. Preserved for reference, not for active use.

## What Belongs Here
- Docs with `status: DEPRECATED` in their frontmatter
- Completed research that produced actionable results (now in other areas)
- Superseded specs and standards

## Rules
- **Archived docs retain their original category code** (e.g., a deprecated `GOV-002` stays as `GOV-002` in this folder)
- **Never delete archived docs** — they are historical record
- **Always add `superseded_by`** in frontmatter when archiving

## Category Codes
| Code | Use For |
|:-----|:--------|
| `ARC` | Archive-specific docs (rare) |
| *(original)* | Archived docs keep their original code |
