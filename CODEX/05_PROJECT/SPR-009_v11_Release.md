### T-028: Docker TLS Updates

Update `docker/Dockerfile`:
- `EXPOSE 443`

Update `docker/docker-compose.yml`:
- Port mapping: `"${HTTPS_PORT:-443}:443"`
- Volume: `certs:/data/certs`

Update `docker/.env.example`:
- Add `HTTPS_PORT=443`
- Add `TLS_ENABLED=true`

### T-030: Production Deployment

1. Push to EC2:
   ```bash
   cd ~/fonzygrok && git pull origin main
   cd docker && docker compose up -d --build
   ```

2. Enable TLS (update `.env`):
   ```env
   TLS_ENABLED=true
   HTTPS_PORT=443
   ```

3. Add port 443 to EC2 security group

4. Verify:
   - `curl https://tunnel.fonzygrok.com/` â†’ server info (valid cert)
   - `curl http://tunnel.fonzygrok.com/` â†’ redirect to HTTPS
   - Connect client â†’ tunnel URL is `https://`
   - Inspector works on `localhost:4040`
   - Metrics endpoint returns data after traffic
